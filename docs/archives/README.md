# Documentation Archive

This directory holds long-form, exploratory, or historical documents that are not part of the primary workflows. Content here is preserved for context and reference, but should not be expanded for routine tasks.

Principles:
- No duplication with active guides — link out instead.
- Prefer summarizing and linking to upstream/official docs.
- Move content back to active areas only when it becomes essential and maintainable.

Examples:
- `iterm-architecture-notes.md` — deep-dive notes, superseded by concise setup/config guides.
