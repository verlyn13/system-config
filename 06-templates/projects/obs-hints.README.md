# .obs Hints (Optional)

Create a `.obs/` directory in your project to provide hints to observers.

Suggested files:

- `.obs/paths.txt` — additional directories to scan
- `.obs/thresholds.json` — overrides for warn/fail thresholds
- `.obs/ignore.txt` — glob patterns to ignore in scans

These hints are optional and should not contain secrets.
